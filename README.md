# 9AI Manager Drone

    this repository contains the source code for the drone server of the 9AI Manager backend

## Development

### Prerequisites

1. [PNPM](https://pnpm.io/installation)
2. [nvm](https://github.com/nvm-sh/nvm)


Please follow these steps to setup your development environment.

1. this project uses nvm to manage node version, run the following command to use the correct version.

```bash
nvm use
```

2. install dependencies

```bash
pnpm install
```

3. add environment variables, refer to the `.env.sample` for necessary variables.

4. run the server

```bash
pnpm dev
```
